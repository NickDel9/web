Σειρές που περιλαμβάνει η ενότητα.

Εισαγωγή στον Προγραμματισμό με Python

Εισαγωγή στις αρχές του προγραμματισμού με χρήση της γλώσσας Python. Με την ολοκλήρωση της σειράς μαθημάτων ο χρήστης θα είναι σε θέση να κατανοεί και να αναπτύει βασικά προγράμματα με τη βοήθεια της γλώσσας προγραμματισμού Python.

Περιεχόμενο του μαθήματος :

  • Θα εφαρμόσουμε τις βασικές τεχνικές σχεδίασης αλγορίθμων και δομημένου προγραμματισμού, καθώς θα εφαρμόσουμε κλασσικούς αλγόριθμους ταξινόμησης και αναζήτησης για υπολογισμούς σε πίνακες και αλφαριθμητικά.
  • Θα υλοποιήσουμε προηγμένα προγράμματα που χρησιμοποιούν αναδρομή και να κατανοούν τη σχέση μεταξύ επανάληψης και αναδρομής.
  • Θα γνωρίσουμε βασικές έννοιες συναρτησιακού και αντικειμενοστρεφούς προγραμματισμού στη γλώσσα Python.

Τιμή σειράς μαθημάτων : 55$ .

Εξωτερικοί σύνδεσμοι : Official Website , Wikipedia

Προγραμματισμός με Java

Εισαγωγή στις αρχές του αντικειμενοστραφούς προγραμματισμού με χρήση της γλώσσας Java. Με την ολοκλήρωση της σειράς μαθημάτων ο χρήστης θα είναι σε θέση να κατανοεί και να αναπτύει πολύπλοκα προγράμματα τα οποία ακολουθούν αντικειμενοστραφή τεχνική με τη βοήθεια της γλώσσας προγραμματισμού Java.

Περιεχόμενο του μαθήματος :

  • Θα αναπτύξουμε ένα πρόγραμμα αντικειμενοστρεφούς προγραμματισμού στη γλώσσα Java, περιλαμβάνοντας τον ορισμό κλάσεων, συναρτήσεις δημιουργίας, μεθόδους, μεταβλητές στιγμιότυπου, στατικές μεταβλητές και αναφορές σε αντικείμενα.
  • Θα χειριστούμε τις βασικές έννοιες και αρχές αντικειμενοστρεφούς προγραμματισμού: ενθυλάκωση (απόκρυψη δεδομένων και κώδικα), κληρονομικότητα και πολυμορφισμό.
  • Θα εφαρμόσουμε τεχνικές αντικειμενοστρεφούς προγραμματισμού, όπως ιεραρχία κλάσεων, επαναχρησιμοποίηση κώδικα, υπερκάλυψη και υπερφόρτωση μεθόδων, πολυμορφική επεξεργασία, δυναμική δέσμευση μεθόδων, αφηρημένες και τελικές κλάσεις και μέθοδοι, και διεπαφές στη σχεδίαση και ανάπτυξη λειτουργικού, συμπαγούς και επεκτάσιμου κώδικα.
  • Θα εφαρμόσουμε τεχνικές χειρισμού εξαιρέσεων που παρέχονται από τη γλώσσα Java.

Τιμή σειράς μαθημάτων : 35$ .

Εξωτερικοί σύνδεσμοι : Official Website , Wikipedia

Προγραμματισμός με C++

Εισαγωγή στις αρχές του αντικειμενοστραφους προγραμματισμού με χρήση της γλώσσας C++. Με την ολοκλήρωση της σειράς μαθημάτων ο χρήστης θα είναι σε θέση να κατανοεί και να αναπτύει προγράμματα τα οποία ακολουθούν αντικειμενοστραφή τεχνική και απαιτούν καλή διαχείρηση δυναμικής μνήμης με τη βοήθεια της γλώσσας προγραμματισμού C++.

Περιεχόμενο του μαθήματος :

  • Θα αναλύσουμε ένα υπολογιστικό πρόβλημα ή διαδικασία με βάση τις δυνατότητες των γλωσσών C και C++.
  • Θα κατανοήσουμε και να γράψουμε κώδικα και εκτελέσιμες εφαρμογές σε C και C++, εφαρμόζοντας καλές πρακτικές ανάπτυξης, καθώς και το κόστος εκτέλεσης του παραγόμενου κώδικα μηχανής κατά τη δημιουργία εφαρμογών σε C / C++.
  • Θα επεκτείνουμε και να βελτιστοποιήσουμε κώδικα γραμμένο σε C++ και θα προτείνουμε εναλλακτικές υλοποιήσεις κάνοντας χρήση πολυμορφισμού, τελεστών, σχεδιοτύπων κλάσεων και συναρτήσεων.
  • Θα χρησιμοποιήσουμε επαγγελματικά εργαλεία ανάπτυξης εφαρμογών υψηλών επιδόσεων για τη συγγραφή κώδικα.

Τιμή σειράς μαθημάτων : 65$ .

Εξωτερικοί σύνδεσμοι : Official Website , Wikipedia

Προγραμματισμός στον Ιστό

Αρχιτεκτονική και πρότυπα του Παγκοσμίου Ιστού. Το πρωτόκολλο HTTP και η χρήση του στις εφαρμογές Ιστού. Η γλώσσα HTML5 για τη σημασιολογική περιγραφή περιεχομένου στον Παγκόσμιο Ιστό. Η γλώσσα CSS για την μορφοποίηση περιεχομένου. Εισαγωγή στη γλώσσα προγραμματισμού JavaScript. Ασύγχρονος προγραμματισμός με χρήση JavaScript. Βασικά APIs της πλατφόρμας Web Browser. Χρήση JavaScript για υλοποίηση προγραμμάτων πελάτη στην πλατφόρμα Web Browser. Εισαγωγή στην πλατφόρμα ανάπτυξης εφαρμογών διακομιστή Node.js. Υλοποίηση εφαρμογών και υπηρεσιών Ιστού στην πλατφόρμα Node.js. Αυτοματοποιημένος έλεγχος υπηρεσιών Ιστού στην πλατφόρμα Node.js. H βαση δεδομένων MongoDB. Πρόσβαση σε MongoDB δεδομένα από την πλατφόρμα Node.js.

Περιεχόμενο του μαθήματος :

  • Θα κατανοήσουμε τις ιδιαιτερότήτες του προγραμματισμού στον Ιστό Θα προγραμματίσουμε χρησιμοποιώντας τεχνικές διατήρησης κατάστασης (state) σε εφαρμογές Ιστού πάνω από HTTP.
  • Θα προγραμματίσουμε σε γλώσσες διαχείρισης περιεχομένου (XML, CSS), καθώς και σε γλώσσες προγραμματισμού πελάτη (JavaScript, Ajax, XSLT).
  • Θα κατανοήσουμε μοντέλα ανάπτυξης εφαρμογών στον Ιστό (REST, client-server, Model-View-Controller).

Τιμή σειράς μαθημάτων : 15$ .

Εξωτερικοί σύνδεσμοι : HTML Official Website , JavaScript Official Website , Wikipedia

Προγραμματισμός με C

Η C είναι σχετικά μικρή γλώσσα. Λίγες δεσμευμένες λέξεις απαρτίζουν το λεξιλόγιό της αλλά παρ΄όλα αυτά θεωρείται μία πανίσχυρη γλώσσα προγραμματισμού πάνω στην οποία αναπτύχθηκαν πολλά λειτουργικά συστήματα. Έχει καλά δομημένες εντολές ελέγχου και ισχυρούς τύπους δεδομένων.

Περιεχόμενο του μαθήματος :

  • Βασικές έννοιες της γλώσσας C: Τύποι, τελεστές και παραστάσεις.
  • Ροή Ελέγχου: Εντολές υπό συνθήκη, Εντολές επανάληψης.
  • Διαδικαστικός Προγραμματισμός: Συναρτήσεις, Δείκτες.
  • Δομές Δεδομένων: Πίνακες, Εγγραφές & πίνακες Εγγραφών
  • Είσοδος και Έξοδος: Αρχεία, Χαρακτήρες, Αλφαριθμητικά. Χρήση του προγραμματιστικού περιβάλλοντος wx-devcpp.

Τιμή σειράς μαθημάτων : 15$ .
